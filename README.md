# SCADA_Process_Monitoring
SCADA_Process_Monitoring

This project simulates an industrial process and demonstrates a complete monitoring workflow:

- Process data generation in Python
- Temperature and pressure simulation
- Engineering threshold-based alerts
- Statistical percentile-based alerts
- Power BI dashboard visualization

Technologies: 
Python |
Pandas |
Matplotlib |
Power BI

Dashboard Features:

✔ Temperature monitoring

✔ Pressure monitoring

✔ Engineering alerts

✔ Statistical alerts

✔ KPI summary cards

✔ Trend analysis
